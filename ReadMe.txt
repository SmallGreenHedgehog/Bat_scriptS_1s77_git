Данную директорию необходимо скопировать в корень отладочной базы.
Структура директории:
-_GIT_
	_Модули
	|	*.ert
	|	
	SRC
	|	_Модули
	|	|	* - декомпилированные модули
	|	_Конфа
	|	|	MD
	|	|	|	* - декомпилированная конфигурация
	edecomp.bat
	ecomp.bat
	mdecomp.bat
	mcomp.bat
	|
	ReadMe.txt
	|
	1cv7.curr.md
	1cv7.new.md
	
Описание файлов и каталогов:

_GIT_/_Модули - каталог для хранения отлаживаемых отчетов (*.ert); 
		отчеты данного каталога декомпилируются с соблюдением структуры в _GIT_/SRC/_Модули;
		в данный каталог компилируются отчеты из _GIT_/SRC/_Модули;
		в _GIT_/_Модули можно создавать репозитории;		
_GIT_/SRC - каталог для хранения в подкаталогах декомпилированных отчетов и конфигураций;
_GIT_/SRC/_Модули - каталог для хранения декомпилированныех отчетов (структура соблюдается);
		в подкаталоги данного каталога декомпилируются отчеты из _GIT_/_Модули с соблюдением структуры;
_GIT_/SRC/_Конфа - каталог для хранения декомпилированной конфигурации;
		конфигурация декомпилируется в подкаталог MD данного каталога;
		в GIT_/SRC/_Конфа можно создавать репозитории;
_GIT_/edecomp.bat - файл скрипта декомпиляции всех *.ert каталога 
	_GIT_\_Модули в _GIT_\SRC\_Модули;
_GIT_/ecomp.bat - файл скрипта компиляции всех исходников (*.ert) из каталога 
	_GIT_\SRC\_Модули в _GIT_\_Модули (с соблюдением структуры);
_GIT_/mdecomp.bat - файл скрипта декомпиляции отладочной базы 
	1cv7.md в _GIT_\SRC\_Конфа\MD (с сохранением 1cv7.curr.md);
_GIT_/mcomp.bat - файл скрипта компиляции конфигурации из 
	_GIT_\SRC\_Конфа\MD в _GIT_\1cv7.new.md;
_GIT_/ReadMe.txt - файл описания назначения скриптов и структуры файлов и каталогов;
_GIT_/1cv7.curr.md - сохраненная конфигурация 1cv7.md отлаживаемой базы;
_GIT_/1cv7.new.md - скомпилированная конфигурация;
