Данную директорию необходимо скопировать в корень отладочной базы.
Структура директории:
-_GIT_
	_Модули
	|	*.ert - отлаживаемые отчеты
	|	
	SRC
	|	_Модули - декомпилированные отчеты (структура соблюдается)
	|	_Конфа
	|	|	MD - декомпилированная конфигурация (1cv7.md)
	|	|	
	edecomp.bat - декомпиляция всех *.ert каталога _GIT_\_Модули в _GIT_\SRC\_Модули
	ecomp.bat - компиляция всех исходников (*.ert) из каталога _GIT_\SRC\_Модули в _GIT_\_Модули (с соблюдением структуры)
	mdecomp.bat - декомпиляция 1cv7.md отладочной базы в _GIT_\SRC\_Конфа\MD (с сохранением 1cv7.curr.md)
	mcomp.bat - компиляция конфигурации из _GIT_\SRC\_Конфа\MD в _GIT_\1cv7.new.md
	|
	ReadMe.txt
